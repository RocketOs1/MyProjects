Project 5
Навзание:
* Парсер данных с веб-страницы.
Суть:
* Программа извлекает информацию с сайта (например, новости, погоду,
курсы валют, цены) и выводит или сохраняет её.
Что изучите:
* HTTP-запросы(requests), парсинг HTML (BeautifulSoup).
Важно:
* Проверяйте robots.txt и уважайте политику сайта.
Усложнение:
* Добавить отправку уведомлений (например, телеграмм-ботом)
при изменении данных. Запланируйте запуск по расписанию.
