Project 10
Название:
* Приложение для автоматизации рутины.
Суть:
* Скрипт, который сам сортирует файлы по папкам, переименовывает 
фотографии по дате, скачивает вложения из почты, заполняет 
шаблонные отчеты.
Что изучите:
* Работа с ОС (os, shutil), автоматизация GUI (pyautogui), работа с почтой
Excel/Word (openpyxl, docx)
